## Overview

A high-converting, mobile-first dark-mode landing page built with pure HTML, CSS, and minimal JavaScript. Designed to feel premium and trustworthy — perfect for early access signups or future CPA/affiliate integration.

### Features
- Premium dark aesthetic with neon gradient accents
- Responsive, thumb-friendly mobile layout
- Early access form (Web3Forms / Formspree ready – no backend needed)
- Pulsing live members counter
- Smooth scroll & subtle hover effects
- 100% static – ideal for GitHub Pages
